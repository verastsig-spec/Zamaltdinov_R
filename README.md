Random Quote Generator

**Автор:** [Рустам Замалтдинов]

Описание

Приложение «Random Quote Generator» — это GUI‑приложение на Python, которое генерирует случайные цитаты из предопределённого списка.

Основные функции:
* Генерация случайных цитат по нажатию кнопки
* Отображение истории сгенерированных цитат (последние 20)
* Фильтрация цитат по автору и теме
* Сохранение истории и цитат в JSON‑файл
* Загрузка данных из JSON при запуске
